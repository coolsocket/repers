# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/)
2.1: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## TL;DR

Engage with the substance, not the person. Disagree respectfully. RePERS is a
harness for repeatable, verifiable agent workflows — treat code reviews and
discussions with the same discipline you'd want applied to a release pack.

## Enforcement

Maintainers may warn, edit, lock, or remove contributions that are abusive,
discriminatory, off-topic, or in bad faith. Repeated or severe violations →
ban from the repository.

To report privately, open a [GitHub Security Advisory](https://github.com/coolsocket/repers/security/advisories/new)
and describe the situation. Response within 72 hours.
